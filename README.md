# manataheri.github.io

Static personal website for Dr. Mana Taheri, served via **GitHub Pages** at
<https://manataheri.github.io/>.

This repository contains **only the published site** — `index.html` (home),
`learning.html`, `innovation.html`, `visual.html`, `contact.html`, and the shared
`styles.css`. GitHub Pages serves these directly from the root of the `main`
branch; there is no build step. The `.nojekyll` file disables Jekyll processing.

The site source and tooling live in a separate project repository.
